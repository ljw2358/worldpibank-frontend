소개 (About)

월드파이뱅크는 Pi 생태계를 위한 초경량 웹앱입니다. 모바일 자동 글자 확대와 이모지로 인한 레이아웃 이슈를 제거해 안정적으로 보이도록 구성했습니다. WorldPiBank is a lightweight web app for the Pi ecosystem, tuned to avoid mobile autosizing and emoji line-height issues.
